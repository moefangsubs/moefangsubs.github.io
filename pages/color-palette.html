COLOR PALETTE
Karena pekerjaan saya berhubungan dengan desain grafis dan juga sinematografi, saya memutuskan untuk membuat halaman ini, yang awalnya dulu hanya gambar saja. Sebuah halaman dimana saya tunjukkan beberapa tangkapan layar MV yang punya color khas, lalu mengekstraknya menjadi 5 warna dalam satu palette. Kalau ada waktu luang, mungkin saya akan tambah lagi palette-nya.

Tinggal buka menu dropdown dibawah tulisan ini, MV mana yang ingin lihat palette-nya. Setelah muncul, nanti di bawah gambar ada code HEX-nya, dan setiap warnanya diberi contoh sederhana bila dikombinasikan dengan warna lain dalam palette yang sama. Untuk menyalin code HEX, tinggal klik saja tulisannya, nanti akan tersalin sendiri.
